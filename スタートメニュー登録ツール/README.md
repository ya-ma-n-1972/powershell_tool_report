# スタートメニュー登録ツール

任意の実行ファイル（.exe / .ps1 / .pyw / .py / .bat など）を、GUIで入力するだけでWindowsのスタートメニューにショートカット（.lnk）として登録するツールです。

## 特徴

- **GUIで簡単登録** - ショートカット名・ターゲット・引数・アイコンを入力するだけ（管理者権限不要）
- **ドラッグ＆ドロップ対応** - フォームにファイルを放り込めばターゲットに設定
- **.ps1 の自動ラップ起動** - `-ExecutionPolicy Bypass -File` でラップ。ウィンドウ非表示／PowerShell 7 (pwsh) 起動を選択可能
- **.py / .pyw の pythonw 起動** - GUIアプリのコンソール窓を出さずに起動
- **拡張子に応じてオプションを自動で有効/無効化** - 今のターゲットに効かない設定はグレーアウト
- **入力チェック付き** - 必須項目・ファイル存在・ファイル名禁止文字の検証、同名ショートカットの上書き確認
- **PowerShellネイティブ** - 追加インストール不要

## クイックスタート

```powershell
# 実行（管理者権限不要）
.\スタートメニュー登録ツール.ps1

# または明示的に
pwsh -ExecutionPolicy Bypass -File ".\スタートメニュー登録ツール.ps1"
```

### 基本的な使い方

1. **ターゲット指定** - 「参照...」かドラッグ＆ドロップで登録したいファイルを選択
2. **名前・オプション設定** - ショートカット名（空なら自動入力）、必要に応じて引数・アイコン・起動オプションを設定
3. **追加** - 「スタートメニューに追加」を押すと `%APPDATA%\Microsoft\Windows\Start Menu\Programs` に `.lnk` を作成

## 対応ターゲットと起動方法

| 拡張子 | 起動方法 |
|--------|----------|
| `.exe` / `.bat` / `.cmd` など | そのままショートカット化 |
| `.ps1` | PowerShell でラップ起動（`-ExecutionPolicy Bypass -File`）。pwsh 優先／ウィンドウ非表示を選択可 |
| `.pyw` | `pythonw.exe` で起動（コンソール非表示） |
| `.py` | 既定は関連付け起動。オプションで `pythonw.exe` 起動（コンソール非表示）に切替可 |

## 起動オプション（チェックボックス）

対象の拡張子に応じて、関係するものだけが有効になります。

- **.ps1 は PowerShell でラップ起動する** … `.ps1` のとき有効
- **ウィンドウを隠す** … ラップ時の下位オプション。GUI/常駐向け。`Read-Host` 等の対話型コンソールでは OFF にする
- **PowerShell 7 (pwsh) で起動する** … ラップ時の下位オプション。pwsh が無ければ Windows PowerShell にフォールバック
- **.py は pythonw.exe で起動する** … `.py` のとき有効。コンソール窓を出さない

## 動作環境

- Windows 10 / 11
- Windows PowerShell 5.1 または PowerShell 7
- .NET Framework / Windows Forms（Windows標準）

## 注意事項

- **.ps1 の文字コード**: BOM なし UTF-8 の `.ps1` は Windows PowerShell 5.1 で日本語が文字化けし起動に失敗することがあります。本ツールは登録時に検出して BOM 付与を提案します（PowerShell 7 で起動する場合は不要なので提案しません）。
- **引数のクォート**: スペースを含む引数は `"..."` で囲んでください。
- **.py のコンソール窓**: `.py`（≠ `.pyw`）は通常 `python.exe` 関連付けで起動するためコンソール窓が出ます。GUIアプリなら「pythonw.exe で起動」を有効にしてください（`pythonw.exe` が見つからない場合は関連付け起動にフォールバックします）。
